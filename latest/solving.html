Solving

Solving systems

solve

HomotopyContinuation.solve — Function.
solve(F::Vector{<:MP.AbstractPolynomial}; options...)

Solve the system F using a total degree homotopy.

solve(G, F, start_solutions; options...)

Solve the system F by tracking the each solution of G (as provided by start_solutions).

source

Depending on the input solve returns one of the following types

HomotopyContinuation.Solving.AffineResult — Type.
AffineResult <: Result

The result of an (non-homogenous) system of polynomials.

source
HomotopyContinuation.Solving.ProjectiveResult — Type.
ProjectiveResult <: Result

The result of a homogenous system of polynomials.
